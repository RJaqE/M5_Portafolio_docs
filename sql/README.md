# Scripts SQL del Proyecto

Esta carpeta contiene el código fuente para la construcción y manipulación de la base de datos "Gestión Académica".

## Orden de Ejecución

Para replicar la base de datos, ejecute los archivos en el siguiente orden estricto:

1. **01_crear_tablas.sql**: Crea la base de datos, las tablas y define las claves foráneas (DDL).
2. **02_manipulacion_datos.sql**: Inserta datos de prueba y contiene ejemplos de actualización/borrado (DML).
3. **03_consultas_reportes.sql**: Contiene las consultas de lectura para generar reportes (DQL).
